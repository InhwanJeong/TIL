정보보호개론

## 0. 개요
- PKI(public key, 공개 키) 기술, 기반구조 -> 비대칭키 방식
  - RSA
- key(비밀번호): 암호화 = 복호화 -> 대칭키 방식
  - DES, AES, 라인델
- 블록 암호
  - ECB, CBC, CFB, OFB, CTR 모드
- 하이브리드 암호 시스템 -> 비대칭 + 대칭

<br>

- 키: 기밀성
- 일방향 해시함수: 무결성(Integrity)
- 메시지 인증 코드: 인증(Authentication)
- 디지털 서명: 인증(Authentication)
- 인증서: 검증
