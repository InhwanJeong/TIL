## 명령 모드  ( : )  <-> ( esc)
- w : 파일을 저장
- q! : 파일 강제 종료
- wq : 파일 저장 및 종료
- s/단어1/단어2 : 단어1을 찾아서 단어2로 바꾸어준다. + /g 시 한줄의 전체 단어1을 단어2로 바꾸어 준다.

## 일반 모드  

#### 이동 

- 이동 : hjkl
- G : 파일의 맨 끝
- shift + G : +번호 -> 해당라인
- / : 단어를 입력하면 해당 단어를 찾아준다. n : 다음 단어, shift + n : 이전 단어
- % : (, {, {의 짝을 찾아준다. (디버깅 할 때 유용)
- gg : 파일의 맨 처음
- w : 다음 단어로
- e : 다음 단어의 끝으로
- b : 이전 단어로
- (){}[] : 큰 단위로 이동
- ctrl + F : 한 화면 밑으로
- ctrl + B : 한 화면 위로
- ctrl + D :절반 화면 밑으로
- ctrl + U :절반 화면 위로

#### 지우기
- 횟수 - 명령 - 대상
- x : 커서의 한 글자를 삭제한다.
- dw : 한 단어를 삭제한다
- d$ : 커서부터 그 이후 한줄 글자를 모두 삭제한다.
- dd : 한줄 전체 삭제
- 2dd : 두줄 전체 삭제

#### undo 명령
- u : 1회 undo 실행
- U : 한줄 undo 실행
- ctrl + r : undo 한 것을 취소

#### 치환(Replace)
- r : 바꾸고 싶은 글자에 커서를 위치 시킨후 r을 누르고 바꾼다.

#### 변환(change)
- cw : 커서부터 글자 끝까지를 삭제한다

## 입력 모드  ( i, o, a ) <-> (esc)
- i : 커서 앞에서부터 입력
- a : 커서 뒤에서부터 입력
- o : 현재 줄 다음 줄 부터 입력
- O : 현재 줄 뒤의 줄 부터 입력

#### 편집
- y 커맨드
```
yy : 한줄 전체 내용 복사
```
